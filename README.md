# LifeMarketers Website

Upgraded multi-page Next.js website for LifeMarketers with a more modern, premium insurance-brokerage design system.

## Pages
- `/` Home
- `/solutions`
- `/carriers`
- `/about`
- `/blog`
- `/contact`

## Branding included
- Uploaded logo file in `public/logo.jpg`
- Family-owned messaging throughout
- Phone number: `(800) 897-5446`
- "Family Owned Since 2002"
- Premium dark-to-light visual system carried across every page

## Run locally
```bash
npm install
npm run dev
```

## Production build
```bash
npm install
npm run build
npm run start
```

## Deploy
This project is structured for platforms that support Next.js, including:
- Vercel
- Netlify
- Render
- any Node hosting with support for `next build` and `next start`

## Notes
- The contact forms are still UI-only in this package and should be connected to your CRM, email provider, or backend.
- Carrier lists and testimonial content are starter content and can be replaced with final approved copy.
